## What You'll Need

- Mac computer
- iPhone with USB cable
- iPhone and Mac on same WiFi network
- Xcode (from App Store)
- Flutter (we'll help you install if needed)
- Backend server running (we'll set this up)

---

## The Process (Overview)

1. **Check prerequisites** - Flutter, Xcode installed
2. **Get Mac's IP address** - So iPhone can connect to backend
3. **Start backend server** - The app needs this running
4. **Update app configuration** - Set Mac's IP address in app files
5. **Connect iPhone** - USB cable, trust computer
6. **Enable Developer Mode** - On iPhone (iOS 16+)
7. **Open Xcode** - For building and debugging
8. **Run the app** - Build and install on iPhone
9. **Test** - Make sure everything works

**Total time:** 15-20 minutes for first-time setup

## Common Questions

### "Do I need to be a developer?"
No! The guides are written for beginners. Just follow the steps.

### "What if something goes wrong?"
Each guide includes troubleshooting sections. Check the Xcode console for specific error messages.

### "Do I need to pay for anything?"
No. Xcode is free. Flutter is free. You can use a free Apple Developer account for testing.

### "Can I use Android instead?"
Yes! The app supports both iOS and Android. The setup is similar, but you'll use Android Studio instead of Xcode.
